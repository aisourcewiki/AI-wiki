Overview

Model inference refers to the process of using a trained machine learning model to make predictions or decision on new input data, where data can be text, images, or numerical data depends on the task being performed by the model. 

Purpose of model inference

Inference can take place in a variety of locations, from small mobile devices to large-scale cloud-based services. The performance of the inference is critical to the performance of overall machine learning application in terms of simplicity, scalability, and flexibility. Recently, a number of tools have surfaced that enable Machine Learning Scientists to swiftly and effortlessly deploy models, making their models more accessible to a wider range of users. Here, instead of introducing production-ready model inference techniques, we are going to introduce two alternatives for creating interactive user interfaces for machine learning model: Gradio and Streamlit.

Gradio

Gradio is a powerful web-based tool that allows users to quickly create UI component for machine learning models. It has a simple syntax and is easy to set up, and can be installed directly via pip. It uses a Python package that is compatible with many machine learning frameworks, such as Pytorch and TensorFlow. Unlike other packages, Gradio can be run within a Jupyter notebook. It is now widely used in companies like Amazon and Cisco, and academic institution like Stanford Medicine. 

However, although Gradio has good documentation on the official website, it can be hard to find examples of specific features. Additionally, Gradio is great for building UI component but less so for creating web interfaces and dashboards for machine learning models.

Examples of How to deploy models using Gradio can be found in these Medium articles (Article 1 and Article 2).

Streamlit

Streamlit, similar to Gradio, is another popular tool used to create web application for machine learning models and data visualizations. It provides an easy-to-use API for building app with Python and is compatible with many major libraries such as OpenCV, Latex, PyTorch, NumPy, Vega-Lit, and more. It is built on the top of several other frameworks, including Flask, Tornado, and ReactJS. Streamlit is also widely among large companies like Google and Uber.

Although it is not difficult to learn, it does requires some time to learn its own syntax. It has limited support for video and animation and unlike Gradio, it doesn't integrate with Jupyter Notebook/ Google Colab. 

Overall, Gradio and Streamlit are both excellent tools for easily and quickly creating web interfaces for machine learning models without needing any technical expertise.
