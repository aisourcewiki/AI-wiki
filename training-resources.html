Intro

When deciding how to fine tune models, there are two main dimensions to consider, and two main choices for each:

  • How to control outputs - Dreambooth vs. native fine tuning
  • Output size - full model vs. LoRA

The resources below are organized according to this framework. Will note that there are several variations of LoRAs (e.g. LoHA) that can also be trained, but will add that later.
